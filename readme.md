### Features
- Contacts
    - [✓] GET all contacts
    - [✓] GET one contact by id
    - [✓] POST - add new contact (name, email, phone)
    - [✓] PUT - update information about contact
    - [✓] PATCH - add contact to favorites
    - [✓] DELETE contact
- Authentication
    - [✓] POST - Signup - registration (name, email, phone)
    - [✓] POST - Login 
    - [✓] GET - Current
    - [✓] GET - Signout
